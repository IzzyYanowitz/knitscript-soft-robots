The basic idea is for two color knitting, but could really be used for any number of colors
it essentially allows you to have any stitch be any color

for each color you want to use you have to knit the whole row (kind of, more on this later).
with a given carrier you go down each needle, if you want that color on that stitch just knit as normal
however, if you don't, you can alternate tucking similar to the alt tuck cast on

if you don't want a color at all in a row, it seems like you should be able to just miss, tuck, turn, miss, tuck the first stitches so you don't have as much ugliness on the wrong side, but i'm not sure if that would be as stable.
