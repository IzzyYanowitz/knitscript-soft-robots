These files are intended as a supplement to Lea Albaugh, Scott Hudson, and Lining Yao. 2019. Digital Fabrication of Soft Actuated Objects by Machine Knitting. In CHI Conerence on Human Factors in Computing Systems Proceedings (CHI 103 2019), May 4–9, 2019, Glasgow, Scotland Uk. ACM, New York, NY, 104 USA. https://doi.org/10.1145/3290605.3300414

For each of four example objects from the paper, we include: an image ([example_name].jpg), the Knitout code ([example_name].k) and the Javascript code used to generate the Knitout code. Here is the full raw JS file for these examples.
